# VoidBorn/EN

![VoidBorn](https://img.shields.io/badge/version-0.5.3-purple)
![License](https://img.shields.io/badge/license-MIT-blue)
![Platform](https://img.shields.io/badge/platform-Windows-lightgrey)

A roguelike survival action game born in the void. A phoenix feather rises from emptiness - a symbol of rebirth and resilience in darkness.

## 🎮 Features

- **Roguelike Survival**: Endless waves of enemies with increasing difficulty
- **Character System**: Multiple playable characters with unique abilities
- **Weapon Arsenal**: Diverse weapons with upgrade paths
- **Relic Collection**: Powerful artifacts to enhance your build
- **Achievement System**: Track your progress and unlock rewards
- **Auto-Updates**: Seamless updates via GitHub Releases
- **Multi-language Support**: English and Russian

## 🌌 Lore

The game centers around a phoenix feather born in the void - a symbol of rebirth and resilience in emptiness. The visual style reflects this theme: dark void backgrounds with phoenix/feather motifs, orange/gold accents representing the phoenix, and purple/cyan cosmic elements representing the void.

## 📦 Installation

### Windows

1. Download the latest installer from [Releases](https://github.com/kalosinegor64-lang/VoidBorn-releases/releases)
2. Run `VoidBorn Setup.exe`
3. Follow the installation wizard
4. Launch VoidBorn from your desktop or Start menu

### Auto-Updates

VoidBorn automatically checks for updates on launch. When a new version is available, you'll be notified and can update with one click.

## 🎯 Gameplay

- **Survive** against endless waves of enemies
- **Collect** gold and experience
- **Upgrade** your weapons and abilities
- **Unlock** new characters and relics
- **Achieve** milestones and earn rewards

# VoidBorn/RU

![License](https://img.shields.io/badge/license-MIT-blue)

Roguelike-экшен на выживание, рожденный в пустоте. Перо феникса, восстающее из небытия — символ возрождения и стойкости во тьме.

## 🎮 Особенности

- **Roguelike-выживание**: Бесконечные волны врагов с возрастающей сложностью
- **Система персонажей**: Множество игровых персонажей с уникальными способностями
- **Арсенал оружия**: Разнообразное оружие с путями улучшения
- **Коллекция реликвий**: Мощные артефакты для усиления вашего билда
- **Система достижений**: Отслеживайте свой прогресс и открывайте награды
- **Автообновления**: Бесшовные обновления через GitHub Releases
- **Поддержка языков**: Английский и русский

## 🌌 Лор (История)

В центре сюжета — перо феникса, рожденное в пустоте, как символ возрождения и стойкости в небытии. Визуальный стиль отражает эту тему: темные фоны пустоты с мотивами феникса/перьев, оранжевые/золотые акценты, представляющие феникса, и фиолетово-голубые космические элементы, олицетворяющие пустоту.
